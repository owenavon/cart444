Principles

Principles

    This style guide provides designers, and non-designers with a standard on how to implement Owen Avon’s brand identity in a uniform manner across various visual mediums. This is achieved by evoking the following principles.

    Scalable

    Emphasis was made to ensure that all fonts, layouts, and components are displayed in a scalable manner where the user can copy the dynamic code. The scalable code includes various breakpoints which are essential for ensuring proper sizing and spacing of elements across various screen sizes.

    Specific

    The style guide is straight to the point, and only includes what is required. The information is broken into six distinct sections. Each section encapsulates an illustrative example, and it’s accompanied code for easy comprehension, and simple implementation. No extra bluff is presented.

    Stylish

    The design langue is reflected through the interactive interface of the web-based design system. The indicated colours, fonts, layouts, and components are exhibited to introduce the user to the language. The style is modern, sleek, and scalable. Need I say more?
