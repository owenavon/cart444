Code for 12 column code.

}

    width: 8.33%;

}

Margins


Each column has a four sided margin of 0.9375rem. The space is indicated in grey.


Spacing (Gutters)


Each column is seperated with a gutter of 0.9375rem. The space is indicated in grey.


Padding


No padding between columns. The columns must always remain "stuck" together.
